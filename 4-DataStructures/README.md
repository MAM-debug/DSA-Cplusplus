# 4. Data Structures

Complex data structures combining multiple linked lists and hierarchical relationships between nodes.

## Files in this Topic

### multi_level_linked_list.cpp
Multi-level linked list structure. Implementation showing relationships between multiple entities (e.g., courses with student lists).

## How to Compile and Run

```bash
g++ -o multi_level_linked_list multi_level_linked_list.cpp
./multi_level_linked_list.exe
```

## Concepts Covered
- Multi-level linked lists
- Nested node structures
- Multiple pointer chains
- Hierarchical data organization
- Complex node relationships
