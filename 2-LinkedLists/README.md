# 2-LinkedLists

Implementations of linked list variants in C++.

## Source Files

- `singly_linked_list.cpp` - singly linked list implementation
- `singly-2.0.cpp` - alternate singly linked list version
- `doubly_linked_list.cpp` - doubly linked list implementation
- `doubly2.cpp` - alternate doubly linked list version
- `circular.cpp` - circular linked list implementation

## Build and Run

```bash
g++ -std=c++17 -g singly_linked_list.cpp -o singly_linked_list.exe
./singly_linked_list.exe
```

Use the same command pattern for any other file in this folder.

## Concepts Covered

- Dynamic node creation and linking
- Singly and doubly traversal patterns
- Circular linking behavior
