# 9-Stack

Stack implementation practice in C++.

## Source Files

- `stack.cpp` - basic stack operations (LIFO)
- `stack-2.0.cpp` - alternate/extended stack implementation

## Build and Run

```bash
g++ -std=c++17 -g stack.cpp -o stack.exe
./stack.exe
```

Use the same command pattern for any other file in this folder.

## Concepts Covered

- Push, pop, and peek operations
- Overflow and underflow handling
- Stack implementation variants
