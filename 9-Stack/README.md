# Stack

Programs based on stack data structure implementations.

## Files

- stack.cpp: basic stack operations (LIFO)
- stack-2.0.cpp: alternate/extended stack implementation

## Compile and Run

```bash
g++ -std=c++17 -g stack.cpp -o stack.exe
./stack.exe

g++ -std=c++17 -g stack-2.0.cpp -o stack-2.0.exe
./stack-2.0.exe
```

## Concepts Covered

- push, pop, and peek operations
- overflow/underflow handling patterns
- stack usage variants
