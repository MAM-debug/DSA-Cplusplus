# 3-Hashing

Hash table examples and collision-resolution strategies.

## Source Files

- `hash_basic.cpp` - basic hashing with modulo indexing
- `hash_linear_probing.cpp` - linear probing
- `hash_quadratic_probing.cpp` - quadratic probing
- `hash_separate_chaining.cpp` - separate chaining
- `hash_double_hashing.cpp` - double hashing

## Build and Run

```bash
g++ -std=c++17 -g hash_basic.cpp -o hash_basic.exe
./hash_basic.exe
```

Use the same command pattern for any other file in this folder.

## Concepts Covered

- Hash functions and indexing
- Open addressing methods
- Chaining-based collision handling
