# 7-Searching-and-Sorting

Implementations of searching and sorting algorithms in C++.

## Source Files

- `binary_search.cpp` - binary search
- `binary2.cpp` - alternate binary search version
- `bubblesort.cpp` - bubble sort
- `insertionsort.cpp` - insertion sort
- `iselectionsort.cpp` - selection sort
- `merge_sort.cpp` - merge sort
- `quick_sort.cpp` - quick sort
- `quick_sort_mid_pivot.cpp` - quick sort (middle pivot)
- `stablevsunstable.cpp` - stability comparison in sorting

## Build and Run

```bash
g++ -std=c++17 -g binary_search.cpp -o binary_search.exe
./binary_search.exe
```

Use the same command pattern for any other file in this folder.

## Concepts Covered

- Binary search on sorted arrays
- Iterative and divide-and-conquer sorting methods
- Stability behavior in sorting algorithms
- Pivot strategy effects in quick sort
