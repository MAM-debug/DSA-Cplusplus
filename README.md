## Build and Run

Use this pattern from inside a topic folder:

```bash
g++ -std=c++17 -g filename.cpp -o program.exe
./program.exe
```

## Notes

- Each topic folder has a local README with file-level details.
- Existing .exe files in folders are compiled outputs.
