# 6. Arrays and Basics

Array operations and fundamental revision exercises. Review of basic DSA concepts.

## Files in this Topic

### array_basics.cpp
Revision of array operations and basic DSA concepts with object-oriented programming applications.

### linear_search.cpp
Linear search implementation example for arrays.

### Note
Binary search implementation is available under [7-Searching-and-Sorting/binary_search.cpp](7-Searching-and-Sorting/binary_search.cpp).

## How to Compile and Run

```bash
g++ -o array_basics array_basics.cpp
./array_basics.exe
g++ -o linear_search linear_search.cpp
./linear_search.exe
```

## Concepts Covered
- Array declaration and operations
- Array traversal and manipulation
- Basic algorithm implementations
- Revision exercises
